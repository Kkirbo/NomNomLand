## Installation

Clone the repository:

```bash
git clone https://github.com/Kkirbo/NomNomLand.git
cd NomNomLand
```

---

## Running the Project


```bash
php -S localhost:8080 -t public
```

The website will then be available at:

```text
http://localhost:8080
```

> Make sure to use port **8080** as shown above.
